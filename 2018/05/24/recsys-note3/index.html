推荐系统冷启动问题

如何在没有大量用户数据的情况下设计推荐系统 => 冷启动问题

  1. 冷启动问题

    • 用户冷启动 => 如何给全新的用户作推荐
    • 物品冷启动 => 如何把新加入的物品推荐给用户
    • 系统冷启动 => 如何在一个新开发的网站上设计开发个性化推荐系统

  2. 常见的冷启动解决方案

    • 作非个性化的推荐：直接按热门排行榜进行推荐

    • 利用用户注册时提供的个人信息：

    • 去除因词根不同造成的同义词

    • 去除因分隔符造成的同义词
    • 让用户主动反馈不合适的词

  3. 基于图的标签推荐算法

    数据结构：

    + 给用户推荐他自己常用的标签
+ 融合以上两个方法，进行线性加权
+ 或者基于图作标签推荐
